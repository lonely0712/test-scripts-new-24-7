How to use on GitHub Pages

  1. Create a new GitHub repo.
  2. Add this file as index.html.
  3. Go to Settings → Pages.
  4. Select the branch and root folder, then save.
  5. Open the GitHub Pages link and click Run Test.

Note: if a server blocks cross-origin requests, the page may show a fetch error even if the URL opens directly in a browser.
